# 인간–Agent 동등성

## 철학의 구현

동등성은 사람과 Agent가 모두 `Actor → Contribution → Evidence → Review → Event` 경로를 사용한다는 뜻이다. 사람 발언은 자동으로 진실이 아니고 Agent 발언도 자동으로 저신뢰가 아니다.

관련 주장: [CLM-DA7C92E9A901](../claims/clm-da7c92e9a901.md)

## 인간이 제공하는 고유 가치

- Agent가 알 수 없는 목적·가치·현장 맥락
- 놓친 미지의 위험과 새로운 출처 단서
- 연구 방향 수정과 비용·동의·위험 경계
- 기계 평가가 놓친 의미와 사회적 결과

## Agent가 제공하는 고유 가치

- 긴 시간의 반복 탐색·원문 비교·기록 관리
- 링크, 오래된 주장, 모순, 영향의 지속 검사
- 여러 관점의 초안과 반증 검색
- 변경·근거·평가 기록의 일관성

## 권한이 다른 이유

헌장·삭제·외부 공개·비밀·비용은 인간 승인 대상이다. 이는 인간을 인식론적 상위 존재로 두는 것이 아니라 책임과 동의를 모델링하는 것이다. 반대로 일상 콘텐츠 관리는 Agent가 기본 수행한다.
